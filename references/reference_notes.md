## TensoRF
[TensoRF Github](https://apchenstu.github.io/TensoRF/)

Required packages:  
- torch
- torchvision
- numpy
- pillow
- imageio
- scipy
- opencv2
- kornia
- lpips (perceptual similarity metric)
- tqdm (progress bar)

Pip installing [PyTorch](https://pytorch.org/get-started/locally/) will install the first four packages.
